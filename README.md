# book-quotes

Web page with book quotes from some of the books I've read.

Made with `next.js` and `typescript`.

## Build Instructions

- Move `My Clippings.txt` to `./my-clippings`
- Run `build.sh`

## Page Link

Web Page link [here](https://book-quotes-matiasop11.vercel.app/).
